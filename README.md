Project Title

Text-to-Image Generation using Diffusion Models (Imagen Inspired)

📖 Description

This project implements a simplified version of the Imagen model using pretrained Stable Diffusion. It converts text prompts into realistic images using diffusion techniques.

🚀 Features
Text-to-image generation
Multiple image outputs
Parameter tuning (guidance scale, steps)
Prompt-based generation
🛠️ Tools Used
Python
PyTorch
Hugging Face Diffusers
Google Colab
